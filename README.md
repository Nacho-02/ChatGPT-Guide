## 🧩 什么是 GPT？

GPT 是由 **OpenAI** 开发的人工智能聊天助手，基于最新的 **大型语言模型（LLM）技术**，能够理解并生成自然语言文本。
无论是写邮件、撰写报告、创作小说，还是编程、翻译、学习新知识，GPT 都能在几秒内完成高质量内容。

目前最先进的模型是 **GPT-4o** 和 **GPT-5（测试中）**，支持多模态交互（文字、语音、图像），具备更高速度与理解力。

## 🚧 为什么国内访问 GPT 困难？

主要原因包括：

1. 🌐 **网络限制**：OpenAI 未对中国大陆地区开放，直接访问 `chat.openai.com` 会被阻止。
2. 📱 **注册门槛高**：需要海外手机号验证。

因此，国内用户常选择镜像服务或 API 接口平台来访问 GPT-4/5 功能。

## 🙋 常见问题（FAQ）

**Q1：GPT 是完全免费的吗？**
A：GPT-3.5 版本免费，GPT-4o / GPT-5 为付费版。镜像网站多提供免费试用额度。

**Q2：镜像网站安全吗？**
A：推荐站点均为 HTTPS 加密。仍建议避免输入个人敏感信息。

**Q3：镜像版和官网数据互通吗？**
A：不互通。镜像调用 API，与 OpenAI 账户无绑定。

**Q4：GPT-4o 与 GPT-5 有何区别？**
A：GPT-5 支持更强的多模态理解（图像、语音、上下文记忆），性能显著提升。
